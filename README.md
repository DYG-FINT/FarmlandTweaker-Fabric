# 耕地调整 
![Fabric](https://img.shields.io/badge/Fabric-1.21.2~1.21.8-orange) 
![License](https://img.shields.io/badge/License-MIT-blue) 

耕地调整是一个针对 Minecraft 耕地机制的增强与可调节模组。

提供了一套完全可配置的耕地逻辑，让你自由决定耕地该如何表现。 

### 安装环境
- 运行环境：服务端需装、客户端可选 
- 依赖：[Cloth Config API](https://www.mcmod.cn/class/2346.html) 
- 联动：[Mod Menu](https://www.mcmod.cn/class/1675.html) 

# 功能总览
### 灌溉调整 
- 自定义耕地寻找水源的三维范围。
- 可指定任意方块作为“水源”。 

### 踩踏调整 
- 调整踩坏耕地的最小跌落高度 
- 独立控制玩家、生物、非生物是否能踩坏耕地
- 控制是否允许踩坏带作物的耕地 
- 设置实体的最小体积判定阈值
- 可选的扩散破坏系统

### 湿度调整 
- 防止有作物的干燥耕地变为泥土（原版行为，可改为“否”进行关闭） 

### 生物调整 
- 允许滑翔撞击破坏耕地
